# Polygon ID

Polygon ID (now operating as Privado ID) is a self-sovereign identity platform built on the iden3 protocol and Polygon blockchain networks. It enables organizations to issue privacy-preserving verifiable credentials (VCs) and allows users to authenticate and share proofs without revealing underlying personal data, using zero-knowledge cryptography.

## APIs

| API | Description |
|-----|-------------|
| [Issuer Node API](https://docs.privado.id/docs/issuer-node/issuer-node-api/introduction/) | Full lifecycle management of DIDs and verifiable credentials via REST |
| [Verifier SDK](https://docs.privado.id/docs/verifier/verifier-overview/) | Off-chain and on-chain ZK proof verification for apps |
| [JavaScript SDK](https://docs.privado.id/docs/js-sdk/js-sdk-overview/) | Browser/dApp identity wallet and proof generation |
| [Flutter SDK](https://docs.privado.id/docs/wallet/flutter-sdk/) | Mobile wallet SDK for Android and iOS |
